Si vous voyez cette page, c'est que l'application n'a pu ouvrir Word.
Causes possibles:
  1. Vous n'avez pas Word d'installé sur votre ordinateur
  2. Vos paramètres de sécurité nous empêche d'accéder à vos programmes installés.
Option 1:
Si vous avez un autre traitement de texte, vous pouvez toujours télécharger un modèle vide, en cliquant ici

Option 2
Si vous avez Word et que vous souhaitez que le document Word soit repli avec vos chants sélectionnés, vous devrez modifier vos paramètres de sécurité afin de permettre à l'application d'accéder à votre programme Word.
Vous conviendrez qu'il est sécutitaire pour votre fureteur internet d'empécher certains sites d'accéder à vos applications installés. C'est pourquoi ce comportement est bloqué tant que vous ne modifiez pas volontairement ces paramètres.
N'ayez crainte, ce paramètre sera effectif seulement pour les sites que vous aurez sélectionné.
 
Étapes à suivre pour nous permettre d'accéder à Word.
 Il suffit d'indiquer à votre fureteur que vous vous considérez notre site comme un site de confiance.

Dans le menu de votre fureteur, cliquez sur "Outils", puis sur "Options internet ..."

Vous aurez l'écran suivant:
Cliquez sur l'onglet "Sécurité"

Cliquez ensuite sur l'icone "Sites de confiance", puis sur le bouton "Sites ..."
Vous aurez alors l'écran suivant:
La plupart du temps, cette page sera vide.
Entrez l'adresse "http://chants.carrefourcc.net", puis sur le bouton "Ajouter" 
Attention: Vous ne pourrez pas cliquer sur le bouton "Ajouter" si la case à cocher plus bas n'est pas décoché.
Cliquez ensuite sur "OK" accepter cet ajout et fermer cette fenêtre Vous serez de retour dans l'écran précédent.
Ensuite, cliquez sur le bouton 4 "Personnalisez le niveau".
C'est à cette étape que vous allez déterminer quels droits vous voulez accorder à vos sites de confiance.
Vous aurez l'écran suivant:
Cette page affiche tous les paramètres de sécurité à considérer pour vos sites de confiance.
Il faut trouver l'item "Contrôles d'initialisation et de script ActiveX non marqués comme sécurisés"
C'est ce paramètre qui nous permet ou non d'ouvrir Word, afin d'y insérer vos chants sélectionnés.
Activer: (Option recommandée) Cette option nous permettra d'ouvrir Word sans problème.
Demander: Vous aurez un message de confirmation avant chaque tentative d'ouvrir Word.
Désactiver: Il nous sera impossible d'ouvrir Word et cette fenêtre réapparaitra.

Faites votre sélection (de préférence "Activer"), puis fermez la fenêtre en cliquant sur "OK".
De retour dans l'écran précédent, faites encore "OK" pour fermer la fenêtre des options Internet.

Réesayez encore une fois d'appuyer sur le bouton "Ouvrir Word" et ca devrait fonctionner.
